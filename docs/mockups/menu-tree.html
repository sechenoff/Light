Что изменилось по сравнению с сегодняшней структурой

+ Новое:
  • "Мой день" — стартовая страница, контент меняется по роли (у кладовщика — что сегодня выдавать, у техника — что чинить, у вас — что согласовать)
  • "МОИ ЗАДАЧИ" — секция action-items с бейджами-счётчиками. То что прямо сейчас ждёт действия именно от этого человека
  • "На согласование" — отдельный экран, видит только руководитель, очередь черновиков (статус DRAFT)
  • "МАСТЕРСКАЯ" — раздел для техника: очередь поломок / в работе / архив. Сейчас сломанное видно только в комментариях сканов
  • "Отклонённые" — для кладовщика: брони, которые вы не одобрили, надо переделать
~ Переупорядочено:
  • Бывший огромный /admin (2 771 строка, 4 вкладки) — разбит на 4 отдельных пункта в разделе АДМИН: Пользователи / Сленг / Импорт прайсов / Настройки
  • "Бронирование оборудования" (сегодня это ссылка на /equipment, путающее имя) → теперь "Каталог → Оборудование"
  • Сканер (сейчас /admin/scanner) → перемещён в "ОПЕРАЦИИ → Сканер штрихкодов"
  • "Календарь" и "Все брони" выделены в свою секцию ОПЕРАЦИИ
− Убрано из главного меню:
  • Текущий /dashboard — заменён на "Мой день" (тот же смысл, новое содержимое)
  • /equipment/import, /equipment/manage, /settings — прячутся внутрь соответствующих экранов как кнопки/подстраницы, не в главном меню
  • Логин-лендинг (сегодня /) — не в меню, это публичная часть
