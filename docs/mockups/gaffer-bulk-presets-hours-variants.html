Сравнение

Вариант Вертикальное место Скорость на мобиле Диапазон значений Плюсы Минусы
Сейчас ~48 px быстро — на частых только 10/12/14/16 или «как-то сам разберись» компактно, привычно «свой» визуально не работает — ловушка
A · инпут ~88 px (когда раскрыт) быстро — цифры клавиатуры любое целое 0-99 точное значение, клавиатурный ввод вертикальный скачок при раскрытии
B · слайдер ~96 px (всегда) средне — нужен точный тап 8-18 непрерывно визуально живо, хороший жест на мобиле, всегда виден диапазон выше; не даст 20ч+ без расширения; тап не всегда попадает
C · комбо ~150 px быстро — на якорях, плавно — между 8-18 + якоря 10/12/14/16 лучший из обоих миров самый высокий, два элемента управления одним значением
