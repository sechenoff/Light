A

«Ремонт» — янтарный 🔧 (рекомендую, как на макете)

Янтарный = «внимание/в работе» по семантическому канону (IBM Plex + спокойная палитра). Зелёный приберегаем только для «Принято», чтобы взгляд не путал «всё ок» и «нужен ремонт».

B

«Ремонт» — зелёный, как вы сказали

Кнопка ремонта зелёная. Тогда для «Принято» возьмём другой маркер (например, синяя галочка/контур), чтобы два зелёных не сливались.
