На макете: «Принять всё» делает всё зелёным; «🔧 Сломано» раскрывает строку с причиной + камерой; «⏳ Не вернули» оставляет единицу у клиента. Выберите модель взаимодействия:

A

Принять всё + сегмент-кнопки в строке (на макете, рекомендую)

По умолчанию всё «Принято». Кнопка «Принять всё» сверху. У каждой строки 4 мини-кнопки исхода — один тап меняет статус. «Сломано»/«Потеряно» раскрывают мини-форму прямо в строке. Меньше всего тапов в типичном случае «всё ок».

B

Тап по строке → нижняя шторка с 4 крупными кнопками

Строки чистые (только название + текущий статус). Тап по строке открывает большую нижнюю шторку с 4 крупными кнопками-исходами. Крупнее цели для пальца, но +1 тап на каждое исключение и закрытие шторки.

C

Свайп по строке (жесты)

Тап = принято/снять; свайп влево = меню проблемы. Самый быстрый для опытного кладовщика, но жесты менее очевидны и капризнее на разных телефонах/чехлах.
