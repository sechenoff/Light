Что изменилось по сравнению со строгой версией

  • Типографика. Cal Sans (geometric display) для заголовков и больших чисел, Satoshi для body, JetBrains Mono для ID/счетов/дат. Раньше всё было в IBM Plex — там был «дизайн-системный» тон, теперь ближе к редакторскому.
  • Сетка фоном. Едва заметный grid 24×24px — подпись coss.com, даёт «техничный» вайб без давления.
  • Угловые квадраты на каждой рамке (corners). Маленький сигнатурный штрих, который делает интерфейс узнаваемым.
  • Наклонный цветной хайлайт с градиентом blue → purple → orange — использован один раз, на «+ 2 409 500 ₽» в блоке чистого результата. Это якорь внимания; не размазываем.
  • Радиусы 3px (практически острые) — жёстче, чем в первой версии (4–6px). Вместе с тонкими линиями даёт «архитектурное» ощущение.
  • Inline-код стиль на референсах (счёт № 2026-0412, бронь № МФ-2026-03-15). Подаёт технические идентификаторы как то, чем они являются — кодом.
  • Цвет кнопок — primary теперь чёрный (var(--ink)), а не синий. Более «осознанно-минимальный» подход, в духе coss.com / Linear.
  • Меньше семантических соусов в фоне карточек KPI: только верхняя полоска-индикатор (2px), сама карточка остаётся белой. Раньше был ещё и soft-тоновый фон.
