Сравнение подходов

A. Умная строка B. Бок о бок C. Набор-первый
Набор всегда виден ✅ Внизу карточки ✅ В правой панели ✅ Весь интерфейс = набор
Скорость добавления 🏆 Поиск + 1 клик 🏆 Поиск + 1 клик 🔄 Выбрать → поиск → клик
AI-вставка ✅ Кнопка → overlay ✅ В правой панели ✅ В тулбаре
Категории 🏆 Чипы + поиск ✅ Чипы (мельче) 💡 Слоты = категории
Ширина карточек ✅ Полная ширина ⚠️ Половина (тесно) ✅ Полная ширина
Мобильная версия ✅ Отлично ❌ Не влезет ✅ Хорошо
