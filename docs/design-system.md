# Дизайн-система Light Rental

## Токены

### Шрифты

| Переменная | Семейство | Когда использовать |
|---|---|---|
| `font-sans` | IBM Plex Sans | Основной текст, интерфейс, формы |
| `font-cond` | IBM Plex Sans Condensed | Надстрочники (`.eyebrow`), бейджи ролей, метки |
| `font-mono` | IBM Plex Mono | Числа в таблицах (`.mono-num`), коды, технические значения |

Базовый размер: **13.5 px**, межстрочный интервал 1.6.

## Принципы

1. **Business-blue, IBM-style.** Строгий деловой стиль: тёмно-синий акцент (#1e3a8a), чистые поверхности, IBM Plex шрифты. Дизайн похож на банковский или B2B-SaaS.
2. **Монохромная база + акцент + семантика.** Большинство интерфейса — серая шкала (`ink`, `surface`, `border`, `slate`). Синий (`accent`) только для ключевых действий. Цветные палитры — только для семантики (роли, статусы).
3. **Все цвета через токены.** Никаких произвольных hex напрямую в CSS или className. Если нужен новый цвет — добавить в `tailwind.config.js`.

---

## Что запрещено

- **Editorial/coss.com стиль** — крупные заголовки с засечками, журнальные отступы, декоративные рамки (см. правило в `feedback_strict_design_canon`).
- **Inline `style={{ color: "..." }}`** — только Tailwind-классы или CSS-переменные.
- **Произвольные hex в className** — вместо `text-[#abc123]` добавь токен.
- **`bg-sky-*` или `bg-blue-*` без причины** — вместо этого `bg-accent` / `bg-accent-bright` / `bg-accent-soft`.

## Шрифтовая иерархия

| Класс | Применение |
|---|---|
| `.eyebrow` | Надстрочники-метки (caps, condensed, `ink-3`) над заголовками секций |
| `.mono-num` | Числа в таблицах, KPI-блоках — для выравнивания по колонкам |
| `text-sm` (основной) | Весь интерфейс — 13.5 px |
| `text-xs` | Мелкие лейблы, подписи |
| `font-semibold` | Заголовки секций, активные пункты меню |
