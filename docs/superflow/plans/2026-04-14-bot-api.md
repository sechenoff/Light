### Task 1.1: Create `botScopeGuard` middleware

**File:** `apps/api/src/middleware/botScopeGuard.ts`

1. Экспортировать `botScopeGuard(req, res, next)`.
2. Считывать ключ из `req.headers["x-api-key"]` или `Authorization: Bearer ...` (переиспользовать логику `apiKeyAuth.extractKey`; вынести её в shared helper `apps/api/src/middleware/extractApiKey.ts` если удобнее, но можно и продублировать — пометить комментарием).
3. Если ключ не начинается с `openclaw-` → `next()`.
4. Если метод `DELETE` → 403 `{ message: "Bot keys are not allowed to delete", code: "BOT_SCOPE_FORBIDDEN" }`.
5. Сверить `method + path` с whitelist (см. design 2.2). Если не прошло — 403 `{ message: "Bot key does not have access to this endpoint", code: "BOT_SCOPE_FORBIDDEN" }`.
6. Иначе → `next()`.

**Важно:** `req.path` внутри глобального middleware — полный путь (например, `/api/bookings/abc`). Не `req.url` (может содержать query).

**Тесты:** `apps/api/src/routes/__tests__/botScopeGuard.test.ts` (vitest):
- no key → next() called
- web-admin-xxx key → next() called (неfrtfт openclaw-)
- openclaw-xxx + DELETE /api/bookings/:id → 403
- openclaw-xxx + GET /api/bookings → next()
- openclaw-xxx + POST /api/users → 403
- openclaw-xxx + POST /api/bookings/draft → next()
- openclaw-xxx + GET /api/finance/debts → next()
- openclaw-xxx + POST /api/warehouse/auth → 403

**Commit:** `feat(api): add botScopeGuard middleware for openclaw-* API keys`

### Task 1.3: Add `computeDebts()` in finance service

**File:** `apps/api/src/services/finance.ts`

В конец файла добавить экспортируемую функцию `computeDebts(options)`:
```typescript
export async function computeDebts(options: { overdueOnly?: boolean; minAmount?: number } = {}) {
  const now = new Date();
  const bookings = await prisma.booking.findMany({
    where: {
      status: { not: "CANCELLED" },
      amountOutstanding: { gt: 0 },
    },
    include: { client: true },
    orderBy: { expectedPaymentDate: "asc" },
  });

  // Сгруппировать по clientId
  const byClient = new Map<string, ClientDebtAccumulator>();
  for (const b of bookings) {
    const amt = new Decimal(b.amountOutstanding.toString());
    const daysOverdue = b.expectedPaymentDate
      ? Math.floor((now.getTime() - b.expectedPaymentDate.getTime()) / 86400000)
      : null;
    const isOverdue = (daysOverdue ?? 0) > 0 || b.paymentStatus === "OVERDUE";

    const acc = byClient.get(b.clientId) ?? {
      clientId: b.clientId,
      clientName: b.client.name,
      totalOutstanding: new Decimal(0),
      overdueAmount: new Decimal(0),
      maxDaysOverdue: 0,
      projects: [],
    };
    acc.totalOutstanding = acc.totalOutstanding.add(amt);
    if (isOverdue) {
      acc.overdueAmount = acc.overdueAmount.add(amt);
      if (daysOverdue && daysOverdue > acc.maxDaysOverdue) acc.maxDaysOverdue = daysOverdue;
    }
    acc.projects.push({
      bookingId: b.id,
      projectName: b.projectName,
      amountOutstanding: amt.toDecimalPlaces(2).toString(),
      expectedPaymentDate: b.expectedPaymentDate,
      daysOverdue: isOverdue ? daysOverdue : null,
      paymentStatus: b.paymentStatus,
      bookingStatus: b.status,
    });
    byClient.set(b.clientId, acc);
  }

  let debts = Array.from(byClient.values()).map((c) => ({
    clientId: c.clientId,
    clientName: c.clientName,
    totalOutstanding: c.totalOutstanding.toDecimalPlaces(2).toString(),
    overdueAmount: c.overdueAmount.toDecimalPlaces(2).toString(),
    maxDaysOverdue: c.maxDaysOverdue,
    bookingsCount: c.projects.length,
    projects: c.projects,
  }));

  if (options.overdueOnly) {
    debts = debts.filter((d) => new Decimal(d.overdueAmount).gt(0));
  }
  if (options.minAmount != null) {
    debts = debts.filter((d) => new Decimal(d.totalOutstanding).gte(options.minAmount!));
  }

  // Сортировка по totalOutstanding desc
  debts.sort((a, b) => new Decimal(b.totalOutstanding).cmp(new Decimal(a.totalOutstanding)));

  const totalOutstanding = debts
    .reduce((acc, d) => acc.add(d.totalOutstanding), new Decimal(0))
    .toDecimalPlaces(2)
    .toString();
  const totalOverdue = debts
    .reduce((acc, d) => acc.add(d.overdueAmount), new Decimal(0))
    .toDecimalPlaces(2)
    .toString();

  return {
    debts,
    summary: {
      totalClients: debts.length,
      totalOutstanding,
      totalOverdue,
      asOf: now.toISOString(),
    },
  };
}
```

Тип `ClientDebtAccumulator` — локальный, не экспортируется.

**Commit:** `feat(finance): add computeDebts() for aggregated client debts`

### Task 1.4: Add `GET /api/finance/debts` route

**File:** `apps/api/src/routes/finance.ts`

Добавить после существующего `GET /finance/dashboard`:
```typescript
const debtsQuerySchema = z.object({
  overdueOnly: z.enum(["true", "false"]).optional(),
  minAmount: z.coerce.number().nonnegative().optional(),
});

router.get("/finance/debts", async (req, res, next) => {
  try {
    // Синхронизируем статусы перед агрегацией (консистентно с /receivables)
    await paymentStatusSyncForAllBookings();
    const query = debtsQuerySchema.parse(req.query);
    const result = await computeDebts({
      overdueOnly: query.overdueOnly === "true",
      minAmount: query.minAmount,
    });
    res.json(result);
  } catch (err) {
    next(err);
  }
});
```

if (body.dryRun) {

  // Опция: найти существующего клиента по имени, если нет — пропустить quote с dummy id.
  const existingClient = await prisma.client.findUnique({

  const clientIdForQuote = existingClient?.id ?? "dry-run-placeholder";

**Проверить:** `quoteEstimate` не пишет в БД (только читает Equipment) — это pure-функция, так что dryRun = вызов quoteEstimate + форматирование ответа. Безопасно.

if (body.dryRun) {
  const existing = await prisma.booking.findUnique({
    where: { id },
    include: { client: true, items: true },
  });
  if (!existing) throw new HttpError(404, "Booking not found");

  const start = body.startDate ? parseBookingRangeBound(body.startDate, "start") : existing.startDate;

    dryRun: true,
    booking: {
      id: existing.id,
      status: existing.status,
      projectName: body.projectName ?? existing.projectName,
      startDate: start,
      endDate: end,
      items: itemsAfter,

        subtotal: estimate.subtotal.toDecimalPlaces(2).toString(),

          unitPrice: l.unitPrice.toDecimalPlaces(2).toString(),

        })),
      },
    },
  });
  return;
