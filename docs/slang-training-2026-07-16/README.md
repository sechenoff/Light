## Что сделано

- **+349 новых алиасов** (source=MANUAL_ADMIN): выучены из пар «заявка → смета» (70 сильных пар по дате+гафферу) + ручной разбор частых термов.
- **324 коллапса неоднозначности**: у термов с несколькими таргетами оставлен доминантный (по частоте в реальных сметах, с семантическими поправками: «си стенды 30» → 30" стенд, «60 панель» → PavoSlim 60, «фрост рама» → Rosco).
- Пример выученного сленга: `хайроллер`→OVERHEAD 58, `панкейк`→Apple Block 5, `трещотка`→стропа стяжная, `мт про`→MC PRO, `12 бб`→Black/Block 12х12, `5200`→52XT, `хорьки/каппы/мотыли`→Matthellini, `бытовые`→Удлинитель PCE.
