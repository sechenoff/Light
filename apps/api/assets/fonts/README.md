# Шрифты для PDF-сметы (pdfkit)

Файлы `DejaVuSans.ttf` и `DejaVuSans-Bold.ttf` — шрифт **DejaVu Sans** ([лицензия](https://dejavu-fonts.github.io/License.html), свободный).

Поставляются в репозитории, чтобы кириллица в PDF отображалась без настройки окружения.  
Переопределение: переменные `SMETA_PDF_FONT_TTF` и `SMETA_PDF_FONT_BOLD_TTF` в `.env`.
